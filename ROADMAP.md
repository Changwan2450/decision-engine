# Roadmap

## Project Memory v1

- Future roadmap item, not part of the current implementation scope.
- Current scope is local retained-memory governance: provenance, status, TTL, supersession/conflict marking, and selective retrieval.
- Graph traversal, semantic dedupe, and decay scoring are explicitly deferred.
- Do not add Neo4j, Cognee, Zep, Mem0, Graphiti, or other external memory services until the local contract proves insufficient.

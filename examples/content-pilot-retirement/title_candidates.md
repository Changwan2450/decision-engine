# Title Candidates

Pilot topic: `50대가 은퇴를 못 하는 진짜 이유`

These are draft Korean YouTube longform title candidates. They should be persona-tested before use.

## Persona Test Result

Source: `persona_test.json`

Top 3 for the next draft:

1. **50대가 은퇴를 못 하는 진짜 이유**
   - Baseline winner. Persona Engine returned this as the primary title candidate.
   - Low trust risk because it does not add unsupported numbers or a controversy frame.
2. **쉰 살이 넘었는데도 일을 놓지 못하는 이유**
   - Stronger viewer-identification variant.
   - Use if the opening wants a softer, less analytical tone.
3. **은퇴 준비가 부족해서만은 아닙니다**
   - Strong myth-bust angle.
   - Use only if the script quickly acknowledges that individual planning still matters.

Avoid for this pilot:

- `50대가 은퇴를 못 하는 진짜 이유 논란`
- `논란 종결`
- `난리난 사연`
- `역대급 사건`

Reason: the persona run exposed high-scoring but generic controversy hooks. They may create clicks, but they work against the social-economic life-explanation position.
